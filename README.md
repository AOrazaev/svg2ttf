svg2ttf
========

svg2ttf converts SVG graphics to TTF format. That can be useful for different
webfont generation tools.

Usage
-----

Install:

``` bash
npm install -g svg2ttf
```

Usage example:

``` bash
svg2ttf fontello.svg fontello.ttf
```
